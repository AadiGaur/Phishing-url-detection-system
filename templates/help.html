Frequently Asked Questions

Phishing is a cyber attack where attackers create fake websites or emails to steal sensitive information like passwords, credit card details, or personal data.
The system analyzes URL features such as length, special characters, domain structure, and security indicators. These features are passed to a trained Machine Learning model (Random Forest) which predicts whether the website is legitimate or phishing.
This system uses the Random Forest algorithm because it performs well on structured datasets and reduces overfitting by combining multiple decision trees.
The system provides approximately 98% accuracy based on the training dataset. However, phishing techniques evolve constantly, so results should be used as guidance rather than absolute confirmation.
